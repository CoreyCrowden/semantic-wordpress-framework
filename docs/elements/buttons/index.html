Button Basics

.button.disabled.primary.secondary.emphasis

To create a button you simply need to apply the button class to any text element. Buttons work best on anchor links and form submit fields.

Disabled Buttons

.button.disabled.primary.secondary.emphasis

Buttons can be put into a disabled state. This changes the styling and physically disables to button from being pressed. The disabled state is achievable with either the disabled class or the form field disabled attribute.
