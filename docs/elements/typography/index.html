Inverting Type Colors

.inverted.normal.container.secondary

The .inverted class can be applied to container with a dark background to reverse the content colors. This inverted color scheme will apply to all child elements until it is reversed by a .normal class.

Note: due to the nature of CSS and in the attempt to minimise file size .inverted and .normal classes can only be nested 3 times (by default) before they stop working. To change the nesting limitation add the variable $color-class-nesting-depth: 3; to your variables file and change the value to you preferred nesting level. Be warned this can bloat the css file dramatically.
