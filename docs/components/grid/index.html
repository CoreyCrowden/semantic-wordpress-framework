Grid

The grid system uses the modern CSS flex display for creating complex layouts.

To initiate the grid system you simply need to apply the grid class to a div element.

Layout Structure & Widths

To create layout structure, width classes can be applied directly to the child element of a grid or along with the parent grid class.

Defining the width or an item

Child element width classes are named as fractions from halves up to sixths. For example two fifths or three quarters.

Defining the width of child items

The child items of a grid can be evenly divide by using the appropriate proportion class alongside the grid class. For example grid halves or menu fifths.

The grid can be divided up into a maximum of sixths.
