This grid system takes advantage of a large range of classes developed to simplify and extend layout control. The flexbox grid system is extended for use with menus. Being call to with the .menu class instead of grid. All additional grid classes should work as expected within the .menu class. reference-title: CSS Tricks Guide to Flexbox reference: https://css-tricks.com/snippets/css/a-guide-to-flexbox/ ---

Grid

The grid system uses the modern CSS flex display for creating complex layouts.

To initiate the grid system you simply need to apply the grid class to a div element.
