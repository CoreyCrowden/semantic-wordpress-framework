Introducing to the grid

The grid system uses the modern CSS flex display for creating complex layouts.

Initiating the grid system

.grid.item

To initiate the grid system you simply need to apply either the grid class or menu class to an element. All immediate child elements will receive the appropriate flex attributes.

By default all child elements of a grid will naturally span to the width of the content with a maximum width of the parent container.

Special classes for controlling the height and width of grid items

.justified.stretched

Applying the classes justified and / or stretched to either the grid element or individual child items will manipulate the box sizing of the child items.

The justified class controls the width of an element while the stretched class controls the height.

Note: You may need to use the class full height on the inner container of item that are stretched to make them fill the height.

The vertical grid

Grids can be both horizontal or vertical. A vertical grid is useful when creating layouts with vertically flowing content. The is especially effective for mobile device layouts.

Creating a vertical grid

.vertical.viewport.height

By default a grid will display horizontally, but by adding the class .vertical to the .grid or .menu classes you can define a vertical grid.

Vertical grids use the same additional classes as a horizontal grid without having to consider the different orientation. For example setting a grid to .justified the child items will horizontally span the full width of both the horizontal and vertical grids.
